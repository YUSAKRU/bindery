This project is licensed under the **GNU General Public License v3.0 (GPL-3.0)**. See [`LICENSE`](LICENSE) at the repo root for the full legal text.

### Why GPL-3.0?

- **Copyleft protection**: We want contributions and derivative works to stay open too. GPL-3.0 requires anyone who uses or modifies the code to share their changes under the same license, preventing closed-source derivatives.
- **Patent protection**: Compared to GPL-2.0, GPL-3.0 adds an explicit patent grant and anti-"tivoization" provisions (against using hardware locks to restrict freedom) — relevant given this is distributed as a mobile/Android app.
- **Predictable licensing for a public codebase**: For now we don't want the code silently reused in a closed-source product. A move to a more permissive license (e.g. MIT/Apache-2.0) down the line would be a separate, deliberate decision.

### Scope

- This license covers the source code in this repository (`src/`, `android/`, build scripts, etc.).
- Third-party dependencies (packages under `node_modules`) are covered by their own licenses; GPL-3.0 only applies to this project's own code.
- Brand assets (logo, icon, the "Bindery" name) may be subject to a separate brand policy; this document covers the software license only.

### Neden GPL-3.0?

- **Copyleft koruması**: Projeye yapılan katkıların ve türetilen çalışmaların da açık kalmasını istiyoruz. GPL-3.0, kodu kullanan/değiştiren herkesin değişikliklerini aynı lisans altında paylaşmasını zorunlu kılar — kapalı kaynaklı türev ürünlerin önüne geçer.
- **Patent koruması**: GPL-3.0, GPL-2.0'a göre açık patent hibesi ve "tivoization" (donanım kilidi ile özgürlüğü kısıtlama) karşıtı hükümler içerir. Mobil/Android dağıtımı düşünüldüğünde bu ek koruma önemli.
- **Kamuya açık bir kod tabanı için öngörülebilir lisanslama**: Şu an için kodun kapalı kaynaklı bir üründe sessizce yeniden kullanılmasını istemiyoruz. İleride daha izin verici bir lisansa (ör. MIT/Apache-2.0) geçiş gerekirse, bu ayrı bir karar olarak değerlendirilecek.
